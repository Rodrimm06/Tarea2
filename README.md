# Tarea 2 CSS y Flexbox
Parte 1: Se removieron márgenes innecesarios, se cambio la tipografía, los enlaces se ven de color negro, los enlaces no tienen subrayado, se removieron las viñetas de la lista.  
Parte 2: El menú quedó centrado horizontalmente en la pantalla, los elementos del menú aparecen en una sola fila, existe espacio uniforme entre cada opción.  
Parte 3: Cambio de color en los enlaces al pasar el cursor, linea inferior en los enlaces al pasar el cursor.  
Cambios: Se agrego un fondo que se activa al pasar el cursor por un enlace, el texto se pone más en negritas cuando se pasa el cursor, se agrego un fondo redondo de color gris a todo el menú.
